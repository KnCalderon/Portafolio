# Portafolio
Portafolio diseñadora UX UI
Landing page profesional desarrollada como portfolio personal, orientada a presentar mi perfil como diseñadora UX/UI, junto con mis habilidades y proyectos destacados.

Este proyecto tiene como objetivo demostrar la integración entre diseño y desarrollo, aplicando principios fundamentales de diseño visual y buenas prácticas en desarrollo web.

Se trabajó con una interfaz clara, moderna y accesible, priorizando la legibilidad, la jerarquía visual y la experiencia del usuario. El diseño sigue una estética minimalista con fondo oscuro y acentos en color azul, permitiendo destacar los elementos clave como llamados a la acción y enlaces.

Se aplicaron principios de alineación, contraste, repetición y proximidad para lograr una estructura ordenada y coherente. Además, se definieron jerarquías tipográficas que facilitan la navegación y comprensión del contenido.

El desarrollo fue realizado utilizando HTML semántico para estructurar correctamente la información, junto con CSS para la estilización visual y organización del layout. Se incorporó un enfoque mobile-first, asegurando que el sitio sea completamente responsive y funcione correctamente en dispositivos móviles, tablets y escritorio.

Este portfolio incluye secciones de presentación personal, acceso a redes profesionales y visualización de proyectos, permitiendo comunicar de manera efectiva la identidad profesional.

Este proyecto refleja mi capacidad para diseñar y desarrollar soluciones digitales funcionales, coherentes y centradas en el usuario.
